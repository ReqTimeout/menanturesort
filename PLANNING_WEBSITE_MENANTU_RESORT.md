**Domain:** menantu-resort.com

## 1. RINGKASAN EKSEKUTIF

Menantu Resort adalah villa produktif eco-resort seluas 3,5 hektar di Cicalengka, Bandung Timur, dikelola oleh Sahid Hotel & Resort. Website ini harus menjadi mesin pencari leads 24/7 yang mengubah pengunjung menjadi investor dengan:

- **Visual wah, megah, premium** — desain villa resort mewah
- **Copywriting storytelling persuasif** — FOMO, urgency, investasi berharga
- **Fitur interaktif** — simulator KPR, kalkulator ROI, virtual tour
- **SEO aggressive** — ranking untuk ratusan keyword properti Bandung
- **Tracking lengkap** — GA4, GTM, Pixel, Remarketing, Enhanced Conversion

### 8.2 Performance Optimization

**SvelteKit SSG (Static Site Generation):**
- Pre-render semua halaman statis
- ISR (Incremental Static Regeneration) untuk konten dinamis

**Caching Strategy:**
```
# Cloudflare Page Rules
menantu-resort.com/* → Cache Everything
menantu-resort.com/api/* → Bypass Cache
menantu-resort.com/artikel/* → Cache 1 jam

# Service Worker (PWA)
- Cache static assets (CSS, JS, images)
- Offline support untuk halaman statis
- Background sync untuk form submissions
```

**Image Optimization:**
- Format: WebP with JPEG fallback
- Lazy loading native (`loading="lazy"`)
- Responsive images (`srcset`)
- BlurHash placeholder untuk lazy load
- Max width: 1920px untuk hero, 800px untuk konten

**Code Optimization:**
- Tree shaking (SvelteKit auto)
- Code splitting per route
- Minify CSS/JS (Vite build)
- Preload critical resources
- DNS prefetch untuk external resources

**Core Web Vitals Target:**
```
LCP: < 2.5s (Largest Contentful Paint)
FID: < 100ms (First Input Delay)
CLS: < 0.1 (Cumulative Layout Shift)
INP: < 200ms (Interaction to Next Paint)
TTFB: < 800ms (Time to First Byte)
```

    value: 1
