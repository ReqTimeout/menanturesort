## 3. Catatan Kredensial dan Keamanan

Jangan menyimpan password, token, atau credential di file Markdown, repository, chat, atau terminal history.

Password yang pernah dibagikan sebelumnya tidak disimpan di dokumen ini.

Untuk akses remote, gunakan SSH key.

File public key lokal yang terlihat:

```text
/Users/maabook/Desktop/menantu-resort.com/id_rsa (2).pub
/Users/maabook/Desktop/menantu-resort.com/id_rsa (1)
```

Pastikan file private key memiliki permission aman sebelum digunakan:

```bash
chmod 600 "/Users/maabook/Desktop/menantu-resort.com/id_rsa (1)"
chmod 644 "/Users/maabook/Desktop/menantu-resort.com/id_rsa (2).pub"
```

Jika `id_rsa (1)` adalah private key dan `id_rsa (2).pub` adalah public key, gunakan:

```bash
ssh -i "/Users/maabook/Desktop/menantu-resort.com/id_rsa (1)" egokkcmq@HOST
```

Ganti `HOST` dengan hostname/IP server.

## 5. Panduan Setup Project Node.js di Server

Contoh struktur project:

```bash
ssh -i "/Users/maabook/Desktop/menantu-resort.com/id_rsa (1)" egokkcmq@HOST
cd /home/egokkcmq
mkdir -p menantu-resort
cd menantu-resort

```

### `src/index.js`

```js
const express = require('express');
const path = require('path');

const app = express();
const port = process.env.PORT || 3000;

app.use(express.static(path.join(__dirname, '../public')));

app.get('/', (req, res) => {
  res.sendFile(path.join(__dirname, '../public/index.html'));
});

app.listen(port, () => {
  console.log(`Menantu Resort running on port ${port}`);
});
```

### `public/index.html`

```html
<!doctype html>
<html>
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Menantu Resort</title>
</head>
<body>
  <h1>Menantu Resort</h1>
  <p>Green Ethnic Eco Resort in East Bandung.</p>

```

### `package.json` scripts

```json
{
  "scripts": {
    "start": "node src/index.js",
    "dev": "nodemon src/index.js"
  }
}
```

Install PM2:

```bash
npm install -g pm2
pm2 start src/index.js --name menantu-resort
pm2 save
pm2 startup
```
